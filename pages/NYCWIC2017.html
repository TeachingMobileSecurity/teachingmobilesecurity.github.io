This site contains information for the Practical LAbs in Security for Mobile Applications (PLASMA) Workshop being held at NYCWIC.

Workshop Summary

Want to learn about mobile Security? This workshop will introduce you to our mobile security modules which are oriented towards developers of all experience levels. We will conduct a brief activity about how to securely store data on a mobile device. More information on the planned activity may be found here.

No security or Android experience is necessary. For a full list of activities, click here

Workshop Requirements

While participants are not required to have any prior mobile/security experience, they should be sure to do the following BEFORE the workshop:

  1. Bring their own laptop: Unfortunately, due to the number of participants we are unable to provide computers to everyone, so please bring your own.
  2. Install Android Studio: Please be sure to install Android studio on your laptop before attending the workshop. You may find details on installing Android Studio here.
  3. Build a Simple App: In order to both ensure that you properly have Android studio setup and configured, and to introduce you to Android, it is useful (but not required) to practice building your first Android app.

Further Activities

Looking for even more educational security activities? The SEED Labs offer more educational security activities in areas such as the web, network, systems, and cryptographic security.
