Activities Access

Introduction

Android activity navigation is quite like how Internet navigation works. In web-based applications, many pages are public. We can navigate to them, simply by knowing the site. In Android, we could do the same thing -- we could navigate to any activity using “am” package commands.

The problem here is that attackers may be able to access restricted activities without authenticating using the "am" package commands.
