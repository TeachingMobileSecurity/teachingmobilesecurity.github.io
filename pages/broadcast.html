Broadcast Messages

Introduction

Broadcast messages are used to announce that an event has happened. They can be used to notify apps that certain system events have occurred, like the fact that the system is now in airplane mode or is in the process of being turned off. Custom broadcast events can also be used by apps to notify others that something has happened (a download being completed, for example).

The security risk here is that apps may intercept broadcast messages or send out fake ones. Depending on the broadcast message itself, it could potentially be some very sensitive information.
