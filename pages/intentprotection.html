Introduction

In Android, we can send data between apps quite easily. For example, when we want to share images on Facebook from a local app, we send the image to the Facebook app over the bundle. Then, the Facebook app reads the image data and posts it.

The problem is that sending data between apps can be insecure in Android. For example, Android will automatically export an Activity for other apps to use if certain requirements are met. See: the Android Developer Guide for more information on this.1
