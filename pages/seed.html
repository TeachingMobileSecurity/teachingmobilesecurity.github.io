Welcome

This site contains information for the Practical LAbs in Security for Mobile Applications (PLASMA) Workshop being held at Syracuse University.

Workshop Summary

Want to learn about mobile Security? This workshop will introduce you to our mobile security modules which are oriented towards developers of all experience levels. We will be conducting a short activity related to secure data storage. No security or Android experience is necessary.

Workshop Requirements

While participants are not required to have any prior mobile/security experience, they should be sure to do the following BEFORE the workshop:

  1. Install Android Studio: Please be sure to install Android studio on your laptop before attending the workshop. You may find details on installing Android Studio here.
  2. Build a Simple App: In order to both ensure that you properly have Android studio setup and configured, and to introduce you to Android, it is useful (but not required) to practice building your first Android app.
