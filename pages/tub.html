Mobile Security Workshop

Welcome

This site contains information for the Practical LAbs in Security for Mobile Applications (PLASMA) Workshop being held at TU Berlin. All related activities may be found at: http://www.TeachingMobileSecurity.com

Workshop Summary

Want to learn about mobile Security? This workshop will introduce you to our mobile security modules which are oriented towards developers of all experience levels. We will be conducting several short activities related to mobile security. The activities will be:

June 29

  1. Properly Using External Libraries
  2. Android, Javascript and Webview
  3. Secure XML Storage

June 30

  1. Secure Data Storage
  2. Secure data transmission (Will require Wireshark.
  3. Color blindness (Accessability)
No security or Android experience is necessary. The workshop will be held in room: MAR0.001

Workshop Requirements

While participants are not required to have any prior mobile/security experience, they should be sure to do the following BEFORE the workshop:

  1. Bring their your laptop: Unfortunately, due to the number of participants we are unable to provide computers to everyone, so please bring your own.
  2. Install Android Studio: Please be sure to install the latest version of Android studio on your laptop before attending the workshop. You may find details on installing Android Studio here.
  3. Build a Simple App: In order to both ensure that you properly have Android studio setup and configured, and to introduce you to Android, it is useful (but not required) to practice building your first Android app.
  4. Install Wireshark: We will be using Wireshark for one fo the activities, so please be sure to install it.
